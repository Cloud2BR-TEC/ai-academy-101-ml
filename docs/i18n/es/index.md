# Machine Learning 101 (Espanol)

Bienvenido a **Machine Learning 101**.

Esta es la version en espanol del contenido principal.

## Por Que Machine Learning 101

- Construir vocabulario comun y confianza en la plataforma.
- Aprender un flujo completo de punta a punta.
- Entender desde datos hasta despliegue.
- Usar explicaciones visuales y practicas.

## Perspectiva Clave

Al final de Machine Learning 101, la idea principal es esta:

- ML no es magia; aprende patrones desde ejemplos.
- Un modelo depende de la calidad de datos y evaluacion.
- ML es un ciclo de vida, no solo entrenamiento.
- Despliegue y monitoreo tambien son parte del trabajo.
