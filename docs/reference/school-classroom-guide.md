# School Classroom Guide

This guide helps deliver Course 101 to school learners using simple language, short activities, and visual explanations.

## Recommended Age Band

- Upper elementary to high school.
- For younger students, prioritize concepts and examples over formulas.

## Teaching Format

- Lesson length: 30 to 45 minutes.
- Explain with one diagram at a time.
- Use one real-life example per concept.
- End each lesson with a 3-question recap.

## Suggested Module Flow

1. Module 01: Machine Learning Basics
: Focus on pattern learning from examples.

2. Module 02: Azure ML Overview
: Explain workflow: problem, data, model, test, deploy.

3. Module 03: Workspace and Authoring
: Present workspace as a project home.

4. Module 04: Assets and Lifecycle
: Explain data, job, model, endpoint as a chain.

5. Module 05: Build Your First Model
: Use one simple dataset and one metric.

6. Module 06: Deploy and Score
: Show endpoint as a question-and-answer service.

7. Module 07: Terraform Foundations
: Explain infrastructure as repeatable setup scripts.

8. Module 08: Fabric and AI Integration
: Explain where data preparation and model building connect.

9. Module 09: Course Wrap-Up and Next Steps
: Use recap questions and a short student presentation.

## Teaching Principles

- Use short sentences and concrete examples.
- Avoid introducing many new technical words at once.
- Reinforce vocabulary every class.
- Keep activities visual and interactive.

## Example Recap Questions

- What is the difference between a feature and a target?
- Why do we split data into training and testing?
- What does an endpoint do?

## Assessment Ideas

- Pair students and ask them to explain one diagram each.
- Give a small dataset and ask what the target should be.
- Ask students to map the lifecycle from data to prediction.
