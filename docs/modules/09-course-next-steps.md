## Course 101 Skills Checklist

- Explain ML basics in plain language.
- Describe core Azure ML assets.
- Train and evaluate a first model.
- Deploy and test a prediction endpoint.
- Use Terraform basics for setup and cleanup.

## Self-Check Questions

- Can I explain feature vs target with a real example?
- Can I explain why train/test split exists?
- Can I explain what endpoint means in one sentence?
- Can I explain one metric (MAE or RMSE) clearly?
