# 05. Build Your First Model

This module adapts the existing machine-learning-model creation workshop into a clear beginner flow.

## Hands-On Sequence

1. Create or select an Azure ML workspace.
2. Create a compute instance.
3. Upload and register the sample dataset.
4. Load data in a notebook and perform basic preprocessing.
5. Train a first model.
6. Evaluate with MAE, MSE, RMSE, and $R^2$.

## Evaluation Reminders

- MAE: average absolute prediction error.
- MSE: squared error emphasis on large misses.
- RMSE: same unit as target variable.
- $R^2$: explanatory power compared with baseline mean prediction.

## Deliverable for 101

A trained model artifact with clear evaluation output and reproducible notebook steps.
