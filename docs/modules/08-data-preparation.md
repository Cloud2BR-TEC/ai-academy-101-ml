# 08. Fabric and AI Integration

Level 101 also introduces how Microsoft Fabric can complement Azure ML workflows.

## Integration Concepts

- Fabric for unified analytics and collaborative data workflows.
- Azure ML for full model lifecycle and deployment control.
- Azure OpenAI and LangChain for LLM-based processing patterns.

## Example Use Cases

- Document summarization and Q&A patterns.
- Batch enrichment workflows with SynapseML.
- Shared data preparation before model training.

## 101 Objective

Understand where integration adds value and where each platform should remain the system of record.
